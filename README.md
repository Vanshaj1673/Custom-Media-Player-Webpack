# Custom Video Player

## Overview

A flexible video player built for modern web applications. It's designed to be lightweight, responsive, and easy to integrate into any project.

## Features

The player comes equipped with everything you need for a great video watching experience:

- **Volume Control**: Adjust audio levels or mute completely, with your preference saved
- **Fullscreen Mode**: Watch videos in fullscreen for an immersive experience
- **Playback Speed**: Speed up or slow down playback depending on your needs
- **Mobile Friendly**: Automatically adapts to different screen sizes for a smooth experience on any device
- **Smart Controls**: Controls appear when needed and hide when you don't need them

## How It Works

The player automatically handles various situations to give you the best experience:

- Controls are hidden while loading and revealed when the video is ready
- Your settings are remembered so you don't have to adjust them every time
- The interface adapts based on what's happening during playback
- Everything is sized and positioned perfectly whether you're on a phone or desktop

## Getting Started

To get the development environment running:

- **Start developing**: `npm start` — launches a local development server
- **Create a release build**: `npm run build` — prepares the final version for deployment
